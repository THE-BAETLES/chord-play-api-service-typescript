

## Chordplay API Server

소프트웨어 마에스트로 13기 Chordplay 프로젝트중 일부인 api 서버입니다.

해당 서버는 Typescript 기반의 Api 서버입니다. 본 프로젝트에서는 Spring 기반의 서버를 사용합니다.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
